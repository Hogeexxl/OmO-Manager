# Data Layer

This directory contains static data files for the application.

## Content

- [providers.json](file:///src/data/providers.json): List of supported AI providers.
- [providers_info.json](file:///src/data/providers_info.json): Detailed metadata for AI providers extraction.
- [models.json](file:///src/data/models.json): Mapping of providers to their supported models.
- [models_info.json](file:///src/data/models_info.json): Flattened detailed metadata for all models.
